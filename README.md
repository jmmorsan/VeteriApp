# 🐾 VeteriApp
Aplicación Android para la gestión integral de una clínica veterinaria.  
Desarrollada con el propósito de facilitar la administración de mascotas, dueños, veterinarios, citas y tratamientos, desde un entorno moderno y accesible.

---

## 📱 Tecnologías empleadas
- **Android Studio (Java)**
- **SQLite + Room** → Persistencia local.
- **JUnit** → Pruebas unitarias.
- **Material Design** → Interfaz moderna y accesible.

---

## 🧠 Funcionalidades
- Gestión de mascotas (CRUD)
- Registro de dueños y veterinarios
- Control de citas y tratamientos
- Módulo memorial para mascotas fallecidas
- Informes e historial médico

---

## 🧩 Arquitectura
Estructura MVVM con uso de DAOs, entidades y repositorios.

---

## 🔮 Futuras líneas de desarrollo
- Integración con **Firebase** para sincronización en la nube
- Sistema de autenticación y roles (veterinario, usuario)
- Notificaciones locales para recordatorios de citas
- Reconocimiento de imágenes para fichas de mascotas
